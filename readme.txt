A powerful mega menu builder with React-based visual editor, column layouts, Bootstrap Icons, AJAX search, and image support.

== Description ==

Mega Menu Builder is a user-friendly and feature-rich WordPress plugin that allows you to design and display custom mega menus. With a clean React-powered drag-and-drop Visual Editor, you can easily organize columns, rows, custom links, images, search fields, and badges to build modern menus.

== Installation ==

1. Upload the `mega-menu-builder` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the "Mega Menu Builder" menu in your WordPress dashboard to start creating custom menus.

== Frequently Asked Questions ==

= Can I use this with any theme? =
Yes! Mega Menu Builder is designed to integrate seamlessly with default WordPress menu walker APIs and popular themes.
